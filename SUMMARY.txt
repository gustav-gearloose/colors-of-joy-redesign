HOW TO USE
----------
1. Open index.html in browser
2. Browse template cards
3. Search or filter as needed
4. Click "View Templates" to explore
5. Use dropdown to switch variations
6. Click "Back to Gallery" to return

HOW TO EXTEND
-------------
Adding a new template is simple:

1. Create HTML file in /templates directory
2. Add entry to templates-config.js:
   {
     id: 'new-template',
     file: 'new-template.html',
     name: 'Template Name',
     description: 'Brief description',
     style: 'YourStyle',
     tags: ['tag1', 'tag2'],
     variations: 5,
     thumbnail: 'image-url',
     featured: false
   }
3. Done! Gallery automatically picks it up.
